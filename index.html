Alberta Climate-Based Seedlot Selection Tool
I Have A Cutblock:
  • Use this tab when you have a cutblock and you are searching for a seed source
  • Species: Select the tree species of interest (currently available for PL, SW, SB, PJ and LT).
  • Seed Zone: Select the Seedzone where for cutblock is located.
  • Go: refreshes the map and tables according to the selections.
I Have A Seedlot:
  • Use this tab when you have a seedlot from a particular seedzone and want to know where it can be used.
  • Species: Select the tree species of interest (currently available for PL, SW, SB, PJ and LT).
  • Seed Zone: Select the Seedzone where for seedlot was collected.
  • Go: refreshes the map and tables according to the selections.
Map:
  • Pan: shift across the map by clicking and dragging on the map
  • Change map scale: Click the +/- buttons or "zoom" by holding the shift key down while clicking and dragging a box shape within the map.
  • Switch Basemap: Select from several options of base maps. Click the window bar to reduce the window.
  • Identify: click on a themed/selected polygon to launch a dialogue box that shows the starting and selected seedzone.
Tables:
  • Sort: clicking a heading will sort the table by that heading, clicking again will change the sort direction

Version 1.0 (2024-07-22):
  1. Using Lat, Elev, MAT, MCMT, TD, MAP, MSP, DD5, trMAP, and MSP in Euclidean distance calculations, where trMAP = MAP^(-1.25), and trMAP = MAP^(-0.75).
  2. Orchards assigned to a SZ on the basis of parent climates
  3. SZ-specific migration distances used.
Version 2.0 (2024-11-27)
  1. Replaced latitude with elevation in the Euclidean distance calculation
  2. Replaced height with height X survival as the response trait in the transfer functions and reduced genetic suitability threshold to 0.90.
  3. Where a plantation SZ (seedzone) had no eligible seed SZ, the genetic suitabililty threshold was reduced to 0.85 to help fill plantation SZ orphans.
  4. Single provincial average migration distance used to simplify analysis.
  5. Additional seedlots added to the seedlot list.
Version 3.0 (2025-02-25)
  1. Focal SZ, seed and plantation SZ, FMA, and FMU layers added.
  2. Genetic suitability threshold dropdown list added.
  3. Labels for seed and plantation SZs added.
  4. The Lt, Pj, and Sb genetic suitability lists use the Sw genetic suitability list as the Sw transfer function was considerably more reliable than the Lt, Pj, and Sb transfer functions.
  5. Dropdown menu to allow selection of migration distance year (2025, 2035, 2045, and 2055) was added.
Version 4.0 (2025-04-25)
  1. Minor typos to CPP layers corrected
  2. Transfer functions modified to use Volume per ha as the response variable and include both ELEV and LAT in the Euclidean distance calculation.
  3. Minimum individual TF R2 for including individual test sites in the pooled transfer function reduced to 0.
  4. New Genetic suitability lists and new seed orchard assignment list generated to address 2 and 3 above.
